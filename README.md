# To-Do List App

A simple command‑line to‑do list application written in Python. Add, remove (by number or name), and clear all tasks with an interactive menu.

## Features
- View your current tasks with numbered list.
- Add a new task.
- Remove a task by its list number or by exact name.
- Clear all tasks (with confirmation).
- Input validation to prevent crashes.

## Requirements
- Python 3.6 or higher (no external libraries needed).

## How to Use
Run the script:
```bash
python todo.py
